# GH-Electron-Elm
[![license][license-image]][license-url]

An Electron App with Elm for handling Monster Damage/Statuses in Gloomhaven in order to help minimize setup time.

## Installation

1. Run `npm install`.

## Development

1. Run `npm run app`.
